## 1. 产品概述
AI聊天工具聚合桌面应用，整合多个主流AI聊天服务，提供统一的访问入口和无缝切换体验。
- 解决用户在浏览器中频繁切换不同AI聊天工具的困扰
- 支持登录状态持久化，避免重复登录

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 无 - 本地应用 | 使用所有AI聊天工具、管理收藏、切换主题 |

### 2.2 功能模块
1. **主界面**: AI工具侧边栏、浏览器视图区、工具栏
2. **设置页面**: 应用设置、主题切换、工具管理

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主界面 | AI工具侧边栏 | 显示所有支持的AI工具，点击切换，支持收藏标记 |
| 主界面 | 浏览器视图区 | 嵌入式WebView显示AI工具网页，支持Cookie持久化 |
| 主界面 | 工具栏 | 前进/后退/刷新/全屏按钮，当前工具信息显示 |
| 设置页面 | 工具管理 | 添加/删除/编辑AI工具配置 |
| 设置页面 | 主题设置 | 冷淡风/深色模式切换 |

## 3. 核心流程
用户打开应用 → 在侧边栏选择AI工具 → WebView加载对应网页 → 登录一次后状态持久化 → 切换工具无需重复登录 → 使用工具栏导航

## 5. 登录持久化方案

### 5.1 技术原理
- Electron 使用 Chromium 内核，与 Chrome 浏览器完全兼容
- 每个 WebView 拥有独立的 Session（或共享 Session）
- Cookie、LocalStorage、IndexedDB 等数据自动由 Chromium 管理

### 5.2 实现方案
1. **Persistent Session**: 为每个 AI 工具创建独立的持久化 Session
2. **Cookie 保存**: 使用 session.defaultSession.cookies API
3. **数据存储路径**: Session 数据存储在用户数据目录
4. **跨会话保持**: 下次打开应用自动恢复登录状态

### 5.3 优势
- ✅ 与浏览器体验完全一致
- ✅ 支持所有登录方式（账号密码、扫码、OAuth等）
- ✅ Cookie、LocalStorage 自动持久化
- ✅ 无需额外开发登录逻辑
